## ⭐️ Feature

- Spring security JWT 인증 방식을 통해 로그인 기능을 구현했습니다.
- 특정 타석을 원하는 시간에 예약 가능한 지 미리 확인할 수 있습니다.
- 예약 시간 10분 전까지는 예약을 취소할 수 있습니다.
- 예약 시간 10이 지난 후에도 입실하지 않은 경우 해당 예약 건은 자동 취소됩니다.
- 나의 예약 및 입/퇴실 등 전체 이용 내역 조회를 할 수 있습니다.
